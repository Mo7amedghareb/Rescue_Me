# Rescue Me — Roadside Assistance Platform

## Overview

**Rescue Me** is a full-stack roadside assistance web platform designed to help drivers during vehicle breakdowns and emergency situations by connecting them with nearby service providers quickly and efficiently.

The platform provides a fast, reliable, and user-friendly experience for requesting roadside assistance services such as tow trucks, mechanics, and car electricians while improving response time and road safety.

---

# Features

* Request roadside assistance services through a simple and intuitive interface
* Real-time connection between drivers and nearby service providers
* User authentication system with sign-up and login functionality
* Service request tracking and live status updates
* Driver and service provider dashboards
* Location sharing for faster assistance
* User feedback and rating system
* Responsive and modern UI/UX design
* Interactive SweetAlert2 pop-ups for a smoother user experience

# System Architecture

The platform is divided into three main parts:

1. **Frontend Client**

   * Handles the user interface and interaction
   * Built using HTML, CSS, JavaScript, and Bootstrap

2. **Backend API**

   * Manages authentication, service requests, providers, and business logic
   * Developed using ASP.NET Core Web API

3. **Database**

   * Stores users, providers, orders, and service data
   * Built with SQL Server

# Vision

Rescue Me aims to simplify roadside assistance by creating a direct connection between drivers and trusted service providers through a modern, accessible, and reliable digital platform.

The goal is to reduce waiting time, improve emergency response efficiency, and enhance the overall roadside assistance experience for drivers.
